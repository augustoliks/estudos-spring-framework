# spring-inject-dependency

Exemplo simples e objetivo do uso do Framework Spring, para implementação de injeção de dependências.

Estrutura do projeto:


```shell
├── mvnw
├── mvnw.cmd
├── pom.xml
├── README.md
└── src/main/java/br/fatec/di
    ├── acoes
    │   └── Acoes.java
    ├── Application.java
    ├── caracteristicas
    │   ├── Brasileiro.java
    │   ├── Japones.java
    │   └── Mestico.java
    ├── dependenciaconfig
    │   └── FabricaPessoas.java
    └── templates
        ├── PessoaBrasil.java
        ├── PessoaJapao.java
        ├── Pessoa.java
        └── PessoaMestico.java
```
